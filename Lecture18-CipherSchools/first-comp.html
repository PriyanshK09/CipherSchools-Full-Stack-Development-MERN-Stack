Adding the First Component

Introduction

In this section, we will add our first React component to the application. Components are the building blocks of React applications and help in creating reusable UI elements.

Creating a Functional Component

Functional components are simple functions that return HTML elements. They are easy to write and understand. Below is an example of a simple functional component:

      
import React from 'react';

function Greeting() {
  return (
    <div>
      <h1>Hello, welcome to our React app!</h1>
    </div>
  );
}

export default Greeting;

In this example, the Greeting component returns a div containing a h1 element.

Using the Component in App.js

Once we have created our component, we can use it in our main App component. Here's how we can do that:

      
import React from 'react';
import Greeting from './Greeting';

function App() {
  return (
    <div>
      <Greeting />
    </div>
  );
}

export default App;

In this example, we import the Greeting component and use it within the App component by including it in the JSX with <Greeting />.
