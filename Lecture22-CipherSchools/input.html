Getting User Input in React

Introduction

In this section, we will learn how to get user input in a React application. Handling user input is a crucial part of building interactive web applications.

Creating a Form Component

We will create a simple form component to get user input and display it. Below is the code for the InputForm component:

      
import React, { useState } from 'react';

function InputForm() {
  const [name, setName] = useState('');

  const handleChange = (e) => {
    setName(e.target.value);
  };

  const handleSubmit = (e) => {
    e.preventDefault();
    alert('The name you entered is: ' + name);
  };

  return (
    <form onSubmit={handleSubmit}>
      <input
        type="text"
        value={name}
        onChange={handleChange}
        placeholder="Enter your name"
      />
      <button type="submit">Submit</button>
    </form>
  );
}

export default InputForm;

In this example, we use the useState hook to manage the state of the input field. The handleChange function updates the state with the value entered by the user, and the handleSubmit function handles the form submission.

Using the Form Component in App.js

Now, we will use the InputForm component in our main App component. Here's how we can do that:

      
import React from 'react';
import InputForm from './InputForm';

function App() {
  return (
    <div>
      <h1>User Input Form</h1>
      <InputForm />
    </div>
  );
}

export default App;

In this example, we import the InputForm component and use it within the App component by including it in the JSX with <InputForm />.
